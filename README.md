# MI Quiz App

Aplikasi quiz untuk mahasiswa D4 Manajemen Informatika yang membantu menguji pengetahuan mereka tentang berbagai topik terkait program studi.

## Features
- 10 pertanyaan tentang Manajemen Informatika
- Sistem scoring
- Penjelasan jawaban
- UI yang responsif
- Mode light dan dark

## Requirements
- Flutter SDK >= 3.7.0

## Installation
1. Pastikan Flutter SDK terinstal
2. Clone repository ini
3. Buka terminal di root project
4. Jalankan `flutter pub get`
5. Jalankan aplikasi dengan `flutter run`

## Usage
- Pilih mode light atau dark
- Mulai quiz
- Pilih jawaban untuk setiap pertanyaan
- Setelah selesai, lihat skor Anda
